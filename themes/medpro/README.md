# Тема MedPro для WordPress

Эта тема предназначена для сайта клиники с кастомным типом записей **Доктор**.  
Использует **Bootstrap 5** для верстки и адаптивности.  

---

## Основные возможности

- Кастомный тип записей **Доктор**  
- Таксономии:
  - **Специализация**
  - **Город**
- Архивы и страницы отдельного доктора с выводом:
  - Фото
  - Рейтинга
  - Стаж работы
  - Цена
  - Краткое описание и основной контент
- Фильтр на архиве докторов:
  - По специализации
  - По городу
  - Сортировка по рейтингу, цене или стажу
- Хлебные крошки для навигации  

---

## Установка

1. Скопировать папку темы в ваш WordPress: themes/medpro -> wp-content/themes/medpro
2. Активировать тему через админку WordPress: **Внешний вид → Темы → Активировать MedPro**

3. Убедитесь, что включены кастомные типы записей и таксономии (они регистрируются автоматически при активации темы).

---

## Настройка

- **Главная страница**: выводит 3 лучших докторов по рейтингу и блок со специализациями.  
- **Архив докторов**: выводит фильтр и карточки докторов.  
- **Страница отдельного доктора**: выводит все данные, рейтинг, фото и контент.

> ⚠️ Если базы данных нет, архив и главная будут пустыми, пока не добавите записи через админку.

---

## Советы по разработке

- Используется **Bootstrap 5**, классы уже есть, дополнительных стилей не требуется.  
- JS: в теме подключен `custom.js` и `bootstrap.min.js`.  
- Метабоксы и фильтры находятся в `inc/metaboxes.php` и `inc/custom-post-types.php`.

## Примечание

- Для работы темы требуется WordPress >= 6.0 и PHP >= 8.0.  
- Если хотите использовать тему с вашей локальной сборкой Docker, убедитесь, что база данных загружена (через `db_init/wordpress_dump.sql`), иначе архивы будут пустыми.
