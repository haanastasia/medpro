# MedPro — локальная сборка WordPress

Простой локальный проект WordPress с кастомным типом записей "Доктор" и таксономиями "Специализация" и "Город", через Docker.

---

## Структура проекта

- `db_init/` — дамп базы данных (`wordpress_dump.sql`), если есть, при первом запуске будет загружен.  
- `themes/` — темы WordPress.  
- `plugins/` — плагины WordPress.  
- `uploads/` — медиа файлы WordPress.  
- `logs/` — логи WordPress.  
- `php.ini` — дополнительные настройки PHP.  
- `docker-compose.yml` — сборка Docker.

---

## Быстрый запуск

1. Перейти в папку проекта
2. docker-compose up -d 
3. Переходим http://localhost/ 

## Работа с базой данных

Если в папке db_init/ есть файл wordpress_dump.sql, он будет автоматически загружен при первом запуске. Сейчас бд есть, заполнена тестовой информацией.

Если файла нет или его удалить, создаётся чистая база для новой установки WordPress.
